# Aptamer-Ferritin Interaction Study

Computational research internship at Fe-rry Lifesciences in collaboration 
with MEDIC, IIT Bombay (November 2025 – Present).

## Project Overview
Designing and screening DNA and RNA aptamers for ferritin binding, combining 
in-silico aptamer generation with molecular docking and MD simulation workflows.

## Work Completed So Far
- Generated 1,000 aptamer sequences (DNA & RNA) using Python
- Expanded screening library to 10,000 DNA and 10,000 RNA aptamer sequences
- Protein-aptamer docking using AutoDock and HDOCK (in progress)
- MD simulations in GROMACS to assess aptamer-ferritin complex stability (in progress)

## Tools & Libraries
Python, Pandas, NumPy, Jupyter Notebook, AutoDock, HDOCK, GROMACS
